# Palworld Breeding Calculator

A fast, free, unofficial Palworld breeding calculator built with Next.js 14 and Cloudflare Pages.

## Features

- Two-way breeding lookup: parent → child and child → parents
- 299 Pal species, 44,850 combos
- Shareable result URLs
- Mobile-friendly UI

## Data Source

Breeding power and combo data derived from community research and Palworld.gg.
